## Como Usar?

### Opção 1 (Automático - Recomendado)
```
Criar lançamento → Escolher "Contas a Pagar" → Salvar
↓
Pronto! Aparece em contas-pagar.html
```

```
Abra: http://localhost:3000/sistema-contabil/menu/sincronizar-contas.html
Clique: "Sincronizar Agora"
↓
Pronto! Todas as contas são criadas
```
