# 🚀 PRIMEIROS PASSOS (Quick Start)

## Sua demanda foi respondida! ✅

> "Como fazer para poblar a tabela contas a pagar baseados na tabela lancamentos?"

**Resposta:** Agora funciona automaticamente quando você cria um lançamento!

---

## ⚡ 3 Passos para Começar

### Passo 1: Iniciar Servidor
Terminal:
```bash
node server.js
```
Aguarde aparecer: "Conectado ao MySQL com sucesso!"

### Passo 2: Criar um Lançamento
1. Abra: `http://localhost:3000/sistema-contabil/menu/lancamentos.html`
2. Preencha:
   - Data: hoje
   - Conta Débito: qualquer uma
   - **Conta Crédito: "Contas a Pagar"** ← importante!
   - Valor: 100
3. Clique: "Salvar"

### Passo 3: Verificar Resultado
1. Abra: `http://localhost:3000/sistema-contabil/menu/contas-pagar.html`
2. Atualize: F5
3. 🎉 Viu a linha aparecer?

---

## 📋 Alternativa: Sincronizar Tudo

Se tiver lançamentos existentes:

1. Abra: `http://localhost:3000/sistema-contabil/menu/sincronizar-contas.html`
2. Clique: "Sincronizar Agora"
3. Veja sendo criadas! ✅
